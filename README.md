# Kitte - Prática DevOps

Este é um projeto referente à matéria de Práticas DevOps do curso de Análise e Desenvolvimento de Sistemas. Neste software simples, é possível relaizar um cadastro prático de um animal de estimação e ver seus dados reunidos. 

## Tecnologias Utilizadas

º Back-End: React
º Front-End: JavaScript
º Banco de Dados: Supabase
º Ambiente de Desenvolvimento: VS Code

## Comandos para rodas o projeto

ª npm install
º npm run build
º criar um arquivo na raiz do projeto chamado ".env" e colocar:
REACT_APP_SUPABASE_URL=[url do bd supabase]
REACT_APP_SUPABASE_ANON_KEY=[chave do bd supabase]
º npm start
